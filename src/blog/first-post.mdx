# はじめに

これは、Astro と Bun、そして MDX を使用して作成されたブログの最初の投稿です。

## なぜこのスタックを選んだのか？

- **Astro**: 静的サイト生成に特化しており、パフォーマンスが非常に高い。
- **Bun**: 高速な JavaScript ランタイムで、開発体験が向上する。
- **MDX**: Markdown の書きやすさと React コンポーネントの柔軟性を両立できる。

```javascript
console.log("Hello, Momiji's Blog!");
```

これから技術的な内容や日常の記録を綴っていく予定です。
